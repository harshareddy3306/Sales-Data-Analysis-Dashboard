# Sales Data Analysis and Visualization Dashboard

## Overview

This project analyzes sales data using Python, SQL, and Power BI to identify business trends and insights.

## Technologies Used

- Python
- SQL
- Power BI
- Microsoft Excel

## Key Features

- Data cleaning and preprocessing using Python
- SQL queries for data analysis
- Interactive Power BI dashboard
- Sales trend analysis and reporting

## Project Structure

- python/ → Python analysis scripts
- sql/ → SQL queries
- data/ → Dataset and reports
- powerbi/ → Power BI dashboard
